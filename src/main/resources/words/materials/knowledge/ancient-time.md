# 古代计时法

## 一、十二时辰制

西周时就已使用。汉代命名为夜半、鸡鸣、平旦、日出、食时、隅中、日中、日昳、晡时、日入、黄昏、人定。又用十二地支来表示，以夜半二十三点至一点为子时，一至三点为丑时，三至五点为寅时，依次递推。

### 十二时辰对照表

| 时辰 | 别名 | 时间 | 描述 |
|------|------|------|------|
| 子时 | 夜半、子夜、中夜 | 23时-01时 | 十二时辰的第一个时辰 |
| 丑时 | 鸡鸣、荒鸡 | 01时-03时 | 十二时辰的第二个时辰 |
| 寅时 | 平旦、黎明、早晨、日旦 | 03时-05时 | 夜与日的交替之际 |
| 卯时 | 日出、日始、破晓、旭日 | 05时-07时 | 太阳刚刚露脸，冉冉初升的那段时间 |
| 辰时 | 食时、早食 | 07时-09时 | 古人"朝食"之时，吃早饭时间 |
| 巳时 | 隅中、日禺 | 09时-11时 | 临近中午的时候 |
| 午时 | 日中、日正、中午 | 11时-13时 | 太阳正中 |
| 未时 | 日昳、日跌、日央 | 13时-15时 | 太阳偏西 |
| 申时 | 晡时、日铺、夕食 | 15时-17时 | 下午时分 |
| 酉时 | 日入、日落、日沉、傍晚 | 17时-19时 | 太阳落山的时候 |
| 戌时 | 黄昏、日夕、日暮、日晚 | 19时-21时 | 太阳已经落山，天将黑未黑，天地昏黄，万物朦胧 |
| 亥时 | 人定、定昏 | 21时-23时 | 夜色已深，人们停止活动，安歇睡眠 |

## 二、二十四时辰制

宋以后把二十时辰中每个时辰平分为初、正两部分，这样，子初、子正、丑初、丑正......依次下去，恰为二十四时辰，同现在一天二十四小时时间一致。

## 三、其他时辰制

### 十时辰制

出现于先秦。昼夜各五分。据《隋书·天文志》，昼为朝、禺、中、晡、夕，夜为甲、乙、丙、丁、戊（后用五更来表示）。

### 五时辰制

其名称为晨明、朏明、旦明、蚤（早）食、宴（晚）食、隅中、正中、少还、铺时、大还、高舂、下舂、县（悬）东、黄昏、定昏。参阅《淮南子·天文训》。

### 百刻制

即把昼夜分成均衡的一百刻。其产生与漏刻的使用有关。可能起源于商代。汉时曾把它改造为百二十刻，南朝梁改为九十六刻、一百零八刻，几经反复，直至明末欧洲天文学知识传入才又提出九十六刻制的改革，清初定为正式的制度。

## 四、日常时间名称

古代不一定具备严格的时间意义，而又常见常用的有关名称亦颇不少。

| 时间 | 名称 |
|------|------|
| 日出时 | 旦、早、朝、晨 |
| 日入时 | 夕、暮、晚 |
| 太阳正中 | 日中、正午、亭午 |
| 将近日中 | 隅中 |
| 偏西 | 昃、日昳 |
| 日入后 | 黄昏 |
| 黄昏后 | 人定 |
| 人定后 | 夜半（或叫夜分） |
| 夜半后 | 鸡鸣 |
| 鸡鸣后 | 昧旦、平明（天已亮） |
| 上餐时间 | 食时或早食（日出后隅中前） |
| 晚餐时间 | 晡时（日昃后日入前） |

## 五、更点制度

### 更

古代把晚上戌时作为一更，亥时作为二更，子时作为三更，丑时为四更，寅时为五更。

| 更次 | 时辰 | 现代时间 |
|------|------|------|
| 一更 | 戌时 | 19时-21时 |
| 二更 | 亥时 | 21时-23时 |
| 三更 | 子时 | 23时-01时 |
| 四更 | 丑时 | 01时-03时 |
| 五更 | 寅时 | 03时-05时 |

### 点

把一夜分为五更，按更击鼓报时，又把每更分为五点。每更就是一个时辰，相当于现在的两个小时，即120分钟，所以每更里的每点只占24分钟。

### 更点应用示例

| 说法 | 现代时间 |
|------|------|
| "四更造饭，五更开船" | 后半夜1时至3时做饭，3时至5时开船 |
| "五更三点" | 早晨5时又72分钟，即6时12分 |
| "三更四点" | 午夜1时又96分钟，即2时36分 |
| "从巳时直杀到未时" | 从上午9时直杀到下午3时 |

## 六、刻

### 漏壶计时

古代用漏壶计时。漏壶分播水壶和受水壶两部。播水壶分二至四层，均有小孔，可滴水，最后流入受水壶，受水壶里有立箭，箭上刻分100刻，箭随蓄水逐渐上升，露出刻数，以显示时间。

### 刻的计算

一昼夜24小时为100刻，即相当于现在的1440分钟。每刻相当于现在的14.4分钟。

| 说法 | 现代时间 |
|------|------|
| "午时三刻" | 中午1时43.2分 |
