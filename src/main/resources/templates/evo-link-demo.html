示例 3: 后端 Java 调用

在 Java 代码中使用工具类:

// 方式 1: 异步模式 (推荐)
String taskId = EvoLinkImageUtil.generateImageAsync(
    "一只可爱的小狗", 
    "16:9"
);

// 轮询查询状态
TaskStatus status = EvoLinkImageUtil.getTaskStatus(taskId);
if (status.isCompleted()) {
    String imageUrl = status.getImageUrl();
}

// 方式 2: 同步阻塞模式
TaskStatus status = EvoLinkImageUtil.waitForCompletion(
    taskId, 
    5000,  // 每 5 秒查询一次
    300000 // 最多等待 5 分钟
);
