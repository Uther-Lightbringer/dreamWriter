📝 测试参数

🔄 工作流程说明

  1. 步骤 1：点击"重新生成提示词"按钮，立即返回任务 ID
  2. 步骤 2：前端显示"等待中..."状态，同时发起长轮询请求
  3. 步骤 3：后端在专用线程池中调用 DeepSeek AI 生成提示词
  4. 步骤 4：长轮询最多等待 30 秒，直到任务完成或超时
  5. 步骤 5：任务完成后，弹出通知框显示成功消息
💡 API 说明：
• 发送请求：POST /api/novels/{novelId}/character-cards/{characterId}/generate-prompt
• 长轮询查询：GET /api/novels/{novelId}/character-cards/tasks/{taskId}/status?timeout=30
• 线程池：characterCardTaskExecutor（核心线程数 2，最大线程数 5）
