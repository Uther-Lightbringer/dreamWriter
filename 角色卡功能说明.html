📝 角色卡功能说明

✨ 新增功能
  • 结构化角色卡管理：支持以数组形式批量添加角色卡
  • 角色卡列表展示：在小说详情页可以查看所有角色卡的列表
  • 角色卡详情查看：点击任一角色卡可查看详细信息
  • 角色卡编辑功能：支持新增、编辑、删除角色卡
  • 编辑模式切换：可开启/关闭编辑模式进行批量管理

📌 数据结构

角色卡数据严格遵循以下 JSON 格式：

[
  {
    "id": "char-001",
    "name": "角色姓名",
    "alternativeNames": ["别名 1", "别名 2"],
    "age": 18,
    "gender": "男",
    "occupation": "学生",
    "appearance": {
      "height": "175cm",
      "hair": "黑色",
      "eyes": "棕色",
      "build": "匀称",
      "distinguishingFeatures": "左眉有一道疤痕"
    },
    "personality": "开朗活泼，善良正直",
    "background": "一个普通的高中生...",
    "relationships": [
      {
        "targetId": "char-002",
        "targetName": "李四",
        "relationship": "好友"
      }
    ],
    "notes": "备注信息"
  }
]

🔧 后端 API接口

1. 批量更新角色卡

POST /api/novels/{novelId}/character-cards/batch

请求体：

[
  {
    "id": "char-001",
    "name": "张三",
    "alternativeNames": ["小明"],
    "age": 18,
    "gender": "男",
    "occupation": "学生",
    "appearance": {
      "height": "175cm",
      "hair": "黑色",
      "eyes": "棕色",
      "build": "匀称",
      "distinguishingFeatures": "左眉有疤痕"
    },
    "personality": "开朗活泼",
    "background": "普通高中生",
    "relationships": [],
    "notes": "备注"
  }
]

返回：

{
  "success": true,
  "message": "角色卡更新成功",
  "data": { /* 更新后的小说对象 */ }
}

2. 获取角色卡列表

GET /api/novels/{novelId}/character-cards/list

返回：

{
  "success": true,
  "characterCards": [
    {
      "id": "char-001",
      "name": "张三",
      "age": 18,
      "gender": "男",
      /* ... 其他字段 */
    }
  ]
}

3. 获取角色卡（原始文本）

GET /api/novels/{novelId}/character-cards

说明：此接口保留用于向后兼容，返回原始的 Markdown 文本格式

💻 前端功能

📋 角色卡列表页面
  • 卡片式展示：每个角色以卡片形式展示关键信息
  • 悬停效果：鼠标悬停时显示操作按钮
  • 点击查看：点击卡片查看角色详细信息
  • 编辑模式：切换编辑模式后可删除或编辑角色
➕ 新增/编辑角色卡表单
  • 基本信息：ID、姓名、别名、年龄、性别、职业
  • 外貌特征：身高、发色、瞳色、体型、显著特征
  • 性格描述：性格特点描述
  • 背景故事：角色的背景故事
  • 人际关系：与其他角色的关系（暂未开放编辑）
  • 备注：其他需要说明的信息
👁️ 角色卡详情视图
  • 分区展示：信息按类别分组展示
  • 格式化显示：HTML 转义确保安全
  • 返回按钮：一键返回列表

🎯 使用流程

  1. 打开小说详情页：访问任意小说的详情页面
  2. 点击角色卡按钮：在章节列表上方找到"👥 角色卡"按钮
  3. 查看角色列表：弹窗显示该小说的所有角色卡
  4. 新增角色卡：点击"➕ 新增角色卡"按钮，填写表单
  5. 查看角色详情：点击任一角色卡片查看详细信息
  6. 编辑/删除：点击"✏️ 编辑模式"开启编辑，然后可编辑或删除角色
  7. 保存修改：所有修改会自动保存到数据库

⚠️ 注意事项

  • 角色 ID 建议采用统一格式，如：char-001、char-002
  • 别名多个时使用逗号分隔
  • 年龄为数字类型，可以为空
  • 外貌特征的所有字段都是可选的
  • 保存时会验证必填字段（ID 和姓名）

🔄 兼容性说明

系统保留了原有的 Markdown 文本格式的角色卡接口，旧的文本格式角色卡仍然可以正常显示。新的结构化角色卡功能使用独立的接口进行数据存储和管理。
